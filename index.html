Nyaruru Modding Tool Suite

A collection of tools for unpacking, modifying, and repacking game assets (Nyaruru).

1. Unpacker Tools

  • Decrypt .data Files Decrypts entire folders of game data archives (e.g., `file2.data`).
  • Convert .index to .json Converts compressed `.index` files into readable `.json` format.
  • Extract All Assets Uses the decrypted data and JSON indexes to extract all game assets into a single ZIP file.

2. Repacker Tools

  • Repack Modified Files Rebuilds game data archives from your folder of modified files and the original JSON indexes.
  • Encrypt .data Files Applies the final encryption to your repacked `.data.modified` files.
  • Pack .index Files Compresses and encrypts your modified `.json` index files back into the game's format.

3. Patcher & Utility Tools

  • Patch Game for New Language Patches core game scripts (`nya_config.js`, `nya_package.js`) to add a new language option.
  • Restore Original Scripts Restores original scripts from the `.js.bak` backups created by the patcher.

Follow the steps in order for the best results. Each tool will open in a new tab.
